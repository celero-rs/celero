In this tutorial, you'll build a complete REST API for a blog with:

- **Categories** — organize posts by topic
- **Posts** — CRUD operations with filtering
- **Relationships** — posts belong to categories
- **Pagination** — paginated listing with query filters
- **Swagger UI** — auto-generated API documentation

**What you'll learn:**
- Project setup with the Celero CLI
- Defining routes and handlers
- Database integration with SeaORM
- Writing migrations
- Query filtering and pagination
- Authentication with JWT
- Integration testing
- Deployment

**Prerequisites:**
- Rust installed ([rustup.rs](https://rustup.rs/))
- Celero CLI installed (`cargo install celero-cli`)
- Basic Rust knowledge

**Time:** ~30 minutes

Let's get started with [Part 1: Project Setup](part-1-setup.md)!
